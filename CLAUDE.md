# EmotionFrame Projektregeln

## KI-Daten Grundsatz (WICHTIG)
- **Alle KI-Features müssen 100% echte Daten liefern**
- Keine erfundenen Zahlen, keine Phantasie-Statistiken, keine Schätzungen als Fakten
- Wenn keine echten Daten verfügbar: klar als "Demo-Daten" markieren
- KI-Empfehlungen müssen auf echten, analysierten Daten basieren
- Lieber "Keine Daten verfügbar" anzeigen als falsche Zahlen

## Sprache
- Gesamte UI auf Deutsch (Schweizer Kontext)
- "Follower" bleibt Follower (nicht "Abonnenten")
- Fachbegriffe wie "Engagement", "Hashtag", "Content" dürfen Englisch bleiben

## Design
- Primary Color: #6C5CE7
- Alle neuen Seiten folgen dem bestehenden Design-System
- Framer Motion für Animationen
- Recharts für Charts
